Panduan Transfer Data Antar Device
1
Device A — Pengirim: Buka modal Sinkron → tab Kirim, centang data yang ingin dikirim, klik Generate Kode.
2
Salin kode yang muncul dengan tombol Salin, atau klik teks kode → Ctrl+A → Ctrl+C.
3
Kirim kode ke Device B via WhatsApp, email, Telegram, Google Keep, USB flash drive, atau media apa pun.
4
Device B — Penerima: Buka modal Sinkron → tab Terima, tempel kode, pilih mode penggabungan.
5
Klik Import & Sinkronkan. Data otomatis masuk dan semua halaman diperbarui!
⚠ Catatan Penting
• Kode Sinkron adalah teks biasa (Base64) — dapat dikirim lewat media apapun.
• Kode tidak terenkripsi — hindari mengirim ke pihak yang tidak dipercaya.
• Mode Timpa akan menghapus semua data lokal. Selalu backup dulu jika perlu!
• Sinkron bersifat satu arah per sesi — dari pengirim ke penerima.
